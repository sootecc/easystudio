[12단계] 메뉴에 스타일 지정하기 : 먼저 메뉴에 스타일을 지정하기 전에 와이어프레임

("구성조건_1) 대한천문대_Wireframe.jpg" 이미지 참고 바람)의 메뉴 모양을 확인합니다.

와이어프레임을 보면 메인 메뉴의 아래쪽으로 메인 메뉴와 같은 너비로 서브 메뉴가 펼쳐지도록 되어 있습니다.

이것을 고려하여 메뉴에 스타일을 지정합니다.  style.css 파일에서 클래스 선택자 .menu 를 찾아서 그 안에

아래와 같이 z-index 속성을 추가하여 메뉴 요소가 화면 상에서 가장 앞으로 보일 수 있도록 코딩 추가해 줍니다.

[z-index 추가 코딩 전]

.menu{            /*메뉴 영역 스타일*/
	float:right;
	height:100px;  /*메뉴 영역 높이*/
	width:800px;  /*메뉴 영역 너비*/	
}

[z-index 추가 코딩 후]

.menu{            /*메뉴 영역 스타일*/
	float:right;
	height:100px;  /*메뉴 영역 높이*/
	width:800px;  /*메뉴 영역 너비*/
	z-index:1;  
}


[중요 참고 사항]

참고로, 메뉴는 슬라이드 다운 메뉴(Slide Down Box Menu)로 구성하도록 하고 있습니다. 슬라이드 다운 메뉴는

메인 메뉴에 마우스를 올렸을 때(Mouse over) 서브 메뉴를 슬라이드 다운(Slide-Down)으로 보여주고,

마우스가 메뉴에서 벗어나면(Mouse out) 슬라이드 업(Slide-Up)되면서 서브 메뉴를 숨겨주는 기능합니다.

메뉴의 모양은 스타일시트에서 지정하며, 움직이는 동작 기능은 자바스크립트와 제이쿼리(jQuery)를 이용하여 구성합니다.

[13단계] 각 메인 메뉴의 스타일 지정하기 : style.css 문서에서 .menu 스타일 아래에 다음의 소스 내용을 추가해 줍니다.

.menu{            /*메뉴 영역 스타일*/
	float:right;
	height:100px;  /*메뉴 영역 높이*/
	width:800px;  /*메뉴 영역 너비*/
	z-index:1;  
}

/* .menu 아래에 다음의 소스 코딩을 추가해 줌 */

.navi{            /*전체 메뉴 스타일*/
	float:right;
	margin-top:30px;
	margin-right:10px;
}
.navi>li{         /*각 메인 메뉴 스타일*/
	float:left;
}
.navi>li>a{       /*각 메인 메뉴의 <a> 요소 스타일*/
	height:40px;
	width:180px;
	line-height:40px; 
	font-size:16px;
	font-weight:bold;
	text-align:center;
	background-color:#0066ad;
	color:#ffffff; 
}
.navi>li>a:hover{ /*<a> 요소에 마우스를 올릴 때 스타일*/ 
	background-color:#30a3f3; 
}

[14단계] 서브 메뉴의 스타일을 지정하기 위해 .navi>li>a:hover 스타일 아래에 다음의 소스 내용을 추가해 줍니다.

	background-color:#30a3f3;

/* .navi>li>a:hover 아래에 다음의 소스 코드 추가해 줌 */

.submenu{         /*서브 메뉴 영역 스타일*/
	width:180px;
	position:absolute;
	/* 14단계 코딩 저장하고, Chrome 실행 후 display:none; 부분을 주석 처리해서 변화를 확인해 봄.
	   인사말 위에 마우스를 올려서 a:hover 스타일도 확인해 봄 */
	/* 즉, display:none; 부분을 주석 처리해서 변화를 확인해 보시기 바랍니다.
               변화 확인 후에는 필히! 주석을 다시 풀어주시기 바람  */
	display:none;
	z-index:1;
}
